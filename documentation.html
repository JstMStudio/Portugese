Sobre

JstMStudio é um pequeno estúdio fundado por Jst Tan. JstMStudio está focado em muitas coisas, desde música, escrita de histórias, desenvolvimento de sites e muito mais. JstMStudio é um pequeno estúdio independente remoto que se concentra em muitas coisas para construir um futuro melhor.

Contato

Deseja entrar em contato com o JstMStudio? Atualmente, a equipe está fornecendo várias maneiras para que todos possam entrar em contato com a equipe. Entre em contato conosco para obter informações importantes, como suporte, feedback, sugestões e relatórios de bugs, e não para conteúdo não importante, como spam. Contribuindo para o JstMStudio

Há muitas coisas que você pode ajudar e contribuir para os projetos do JstMStudio e ajudar o twam. Cada uma de suas ajudas e contribuições para nossos projetos são apreciadas por nós, não importa se é apenas um pequeno erro de digitação ou um erro da equipe.



Relatando erros

Uma forma de contribuir com o projeto sem conhecimento técnico é relatar bugs para a equipe. Se você encontrou um bug em nosso site ou software, crie um problema no Github ou envie-nos um e-mail e relate-o. Isso nos ajudaria muito melhorando nosso software para que todos pudessem usá-lo com mais facilidade.

Se você pudesse, também poderia nos ajudar a corrigir bugs relatados quando ainda não consertamos os bugs. Certamente ajudaria muito a equipe por economizar tempo e esforço na localização de bugs. Depois de terminar a programação, você pode enviar um pull request para um de nossos voluntários aprovar e implantar também.



Adicionando conteúdo ao software JstMStudio

Você pode contribuir com o projeto adicionando algum conteúdo ao nosso software. Você pode enviar sites ou projetos para nosso mecanismo de pesquisa, adicionar vídeos, músicas e livros à nossa biblioteca de conteúdo para que todos possam desfrutar e etc.



Códigos

Se você é um desenvolvedor, também pode nos ajudar a melhorar nosso software, como melhorar nossa interface do usuário, ajudar a desenvolver e construir nossos próximos novos recursos e projetos com a equipe e etc. Você também pode ajudar e contribuir para criar mais funções para nosso serviços atuais também.

Doar

JstMStudio é financiado por pessoas como você. Não vendemos nenhum dado do usuário, não enviamos spam para sua tela com anúncios nem bloqueamos itens no paywall. Nós não e achamos isso irritante. Somos financiados através da venda de sites e doações. Se você gostaria de apoiar o projeto, por favor, doe um dólar ou dois para nós. Pode significar menos para você, mas significa muito para nós.

Atualmente, fornecemos dois métodos para que todos possam ajudar, apoiar e doar para nós. Você pode doar para nós via Buy Me A Coffee e Ko-Fi. Você pode nos apoiar doando uma vez ou mensalmente também.

Extensão

JstMStudio Profile é o primeiro site social que realmente oferece suporte para extensão no perfil e nas postagens dos usuários. Uma extensão é um complemento de software instalado em um programa, aprimorando seus recursos. Os usuários podem escolher entre uma variedade de extensões para usar em seus perfis e postar na biblioteca de extensões de perfil do JstMStudio. A extensão JstMStudio Profile é construída pela equipe oficial e pela comunidade.

Os usuários podem instalar todos os plug-ins aqui em seu código, adicionando-o antes do final dos elementos da postagem ou em seu perfil. Instale o plug-in copiando e colando o código abaixo.



Aplausos

Clap é uma das extensões oficiais do Perfil JstMStudio. Ele pode ser usado na postagem dos usuários do perfil JstMStudio. Os usuários podem bater palmas para os usuários postarem quando este plug-in for instalado.



Comentário

O comentário é uma das extensões oficiais do Perfil JstMStudio. Os usuários podem adicioná-los à sua postagem para que o visualizador comente. Os usuários também podem usá-lo para obter feedback sobre suas postagens.



Gosto

Like é uma das extensões oficiais do Perfil JstMStudio. Os usuários podem adicioná-los à postagem para que o visualizador e o escritor mostrem sua reação à postagem. Os usuários também podem reagir para mostrar seu apoio.



Compartilhar

Share é uma das extensões oficiais do Perfil JstMStudio. Os usuários podem adicioná-los à sua postagem para que o visualizador compartilhe sua postagem nas mídias sociais. Os usuários também podem reagir para mostrar seu apoio compartilhando a postagem.



Para cima para baixo

Updown é uma das extensões oficiais do Perfil JstMStudio. Os usuários podem adicioná-los à postagem para que o visualizador e o escritor mostrem sua reação à postagem. Os usuários também podem reagir para mostrar seu apoio. Updown também pode ser usado para enquetes.

API

JstMStudio forneceu muitos serviços e recursos dentro de nosso software de plataforma, JstMStudio App Platform. A API JstMStudio é uma interface de programação de aplicativos para a plataforma de aplicativos JstMStudio que fornece suporte para todos os desenvolvedores implementarem serviços como a plataforma de aplicativos JstMStudio em seu site sem desenvolver os próprios recursos de forma totalmente gratuita e sem taxas. Ele foi desenvolvido para uso de todos e não para ganhar dinheiro de usuários como desenvolvedores e proprietários de sites.

JstMStudio API é uma API de código aberto (Application Programming Interface). Todos podem visualizar, ler e estudar todos os códigos-fonte da API do JstMStudio em nosso repositório oficial do Github. A API JstMStudio é construída usando HTML, CSS e JavaScript vanilla diferente de qualquer API tradicional também e licenciada sob AGPL 3.0.
